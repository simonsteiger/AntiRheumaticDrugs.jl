The drug-class & route taxonomy ​

Each AntiRheumaticDrug is tagged with a class type parameter C <:DrugClass, recovered with category. The class tree nests from broad to specific:

  • Cortisone — corticosteroids (not a DMARD)

  • csDMARD — conventional synthetic DMARDs

  • btsDMARD — the biologic-or-targeted-synthetic umbrella, splitting into

    • bDMARD — biologics: TNFi, CD20i, IFNi, CD28i, BAFFi, and the interleukin family ILi (IL1i, IL5i, IL6i, IL17i, IL23i, IL12_23i)

    • tsDMARD — targeted synthetics: JAKi, PDE4i

category returns the finest node. Two projections climb the tree:

julia
using AntiRheumaticDrugs

ada = classify("L04AB04")
(category(ada), drug_class(category(ada)), mode_of_action(category(ada)))
(TNFi, bDMARD, TNFi)

Use the *_symbol helpers for stable Symbols, or pretty for a display string:

julia
(class_symbol(ada), moa_symbol(ada), pretty(category(ada)))
(:bDMARD, :TNFi, "TNFi")

Routes form a parallel tree rooted at AbstractRoute: Systemic versus the Local routes (Topical, Ophthalmic, Nasal, Inhaled, Intestinal, Rectal).

julia
(route_of(classify("L04AB04")), route_of(classify("D07AC01")))
(Systemic(), Topical())
