AntiRheumaticDrugs.AbstractRoute Type
julia
AbstractRoute

Root of the administration-route taxonomy. A drug's route is read with route_of; is_systemic tests for Systemic. Concrete routes are zero-field singletons (Systemic(), Topical(), …).

source
AntiRheumaticDrugs.AntiRheumaticDrug Type
julia
AntiRheumaticDrug{C<:DrugClass}

A registered antirheumatic substance, tagged at the type level with its drug class C. Fields: name, atc (WHO ATC code), brands::Vector{String}, and route::AbstractRoute. Recover the class with category, the substance name with substance, and the route with route_of. Look one up by ATC code with classify. Implements the DrugInterface AbstractAntiRheumaticDrug contract.

AntiRheumaticDrugs.DrugClass Type
julia
DrugClass

Root of the antirheumatic drug-class taxonomy. Every concrete AntiRheumaticDrug carries a class type parameter C <: DrugClass, recoverable with category. The tree exposes two projection levels: a coarse class node via drug_class (one of Cortisone, csDMARD, bDMARD, tsDMARD) and a finer mode-of-action node via mode_of_action.

AntiRheumaticDrugs.classify Method
julia
classify(atc::AbstractString) -> AntiRheumaticDrug

Look up the registered drug for an ATC code. Throws KeyError if the code is not registered — use try_classify for a nothing-returning variant, or is_registered to test first. Legacy (pre-WHO-revision) ATC codes are registered as aliases of their current code.

Examples

julia
julia> substance(classify("L04AB04"))
"Adalimumab"

julia> moa_symbol(classify("L04AB04"))
:TNFi
source
AntiRheumaticDrugs.count_modes_of_action Method
julia
count_modes_of_action(xs) -> Int

Number of distinct b/tsDMARD modes of action among the drugs xs. csDMARDs and cortisone are ignored — only is_btsdmard drugs are counted.

Examples

julia
julia> count_modes_of_action([classify("L04AB04"), classify("L04AF01")])  # TNFi + JAKi
2

julia> count_modes_of_action([classify("L04AB04"), classify("L04AB01")])  # both TNFi
1
source

true if x's category is a subtype of T. The generic engine behind the class predicates is_cortisone, is_csdmard, is_bdmard, and is_tsdmard.

The abstract supertypes that anchor the interface are also re-exported:

DrugInterface.AbstractDrug Type
julia
AbstractDrug

Root of the drug type lattice. Every concrete drug type subtypes AbstractDrug (directly or through a more specific abstract type).

Subtypes must implement the universal interface:

  • substance

  • mode_of_action

See also AbstractAntiRheumaticDrug.

source
DrugInterface.AbstractAntiRheumaticDrug Type
julia
AbstractAntiRheumaticDrug <: AbstractDrug

Anti-rheumatic branch of the lattice. In addition to the AbstractDrug interface, subtypes must implement the four classification primitives:

  • is_csdmard

  • is_bdmard

  • is_tsdmard

  • is_cortisone

The derived predicates is_btsdmard and is_dmard come for free from these primitives.
